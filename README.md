# Fractured-Wiki
*This repo is used to create the Wiki for my Luanti Rust-Style Game. This is not a full recreation of Rust, just a rust-style voxel game. There are two ways to play Fractured — either using Luanti and downloading "Fractured" from ContentDB and/or downloading what I made for Linux/Windows/MacOs and Android. Fractured was built out of pure curiousity because I accidently discovered Luanti seraching for the Lua Programming Language. All code in the repo is free for you to modify and use, this is just a portfolio project*
